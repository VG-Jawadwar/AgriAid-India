## 📌 Project Overview

**AgriAid India** is a farmer-friendly mobile application developed using **Flutter** that helps farmers easily discover and apply for various **government agricultural schemes**.  

Many farmers are unaware of schemes or face difficulty understanding the application process. AgriAid India solves this problem by organizing schemes in a simple and accessible format.

The application provides:

- ✅ Easy access to government schemes  
- ✅ Clear eligibility and requirement details  
- ✅ Direct redirection to official government portals  
- ✅ Language support (English & Marathi)  
- ✅ Simple UI designed for rural accessibility  

---

## 🚀 Key Features

🌱 **Scheme Categories**  
Schemes are organized into multiple categories for easy navigation.

📋 **Detailed Scheme Information**  
Each scheme includes:
- Description
- Application date
- Required documents
- Applicable states

🔗 **One-Click Apply**  
Users can directly open the official government website to apply.

🌐 **Multi-Language Support**  
Farmers can read scheme details in:
- English
- Marathi

📱 **Lightweight Application**  
The app is built **without a backend database**, making it fast and easy to use.

## 📲 Application Flow
```text
Splash Screen
      ↓
Login / Sign Up
      ↓
Enter Aadhaar Number
      ↓
Home Page
      ↓
Select Scheme Category
      ↓
View Available Schemes
      ↓
Open Scheme Details
      ↓
Click "Apply Now"
      ↓
Redirect to Official Government Website
```
